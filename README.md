# 🚀 AI Resume Analyzer

AI Resume Analyzer is a web application built using React.js and Spring Boot that helps users upload resumes and generate AI-powered interview questions.

## Features

* Upload Resume (PDF)
* Resume Dashboard
* AI Interview Questions Generator
* ATS Score Section
* Resume History Module
* Modern Responsive UI
* Dark Mode Interface

## Tech Stack

### Frontend

* React.js
* Tailwind CSS
* React Router
* Axios

### Backend

* Spring Boot
* Java
* REST APIs

### Database

* MySQL

### AI Integration

* Gemini API / Groq API
