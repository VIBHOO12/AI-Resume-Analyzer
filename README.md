# 🚀 AI Resume Analyzer

AI-powered Resume Analyzer built with React.js and Spring Boot that helps users analyze resumes and generate interview questions using AI.

## Features

✅ Resume Upload

✅ Resume Text Extraction

✅ AI Resume Analysis

✅ AI Interview Question Generation

✅ Modern Responsive UI

🚧 User Authentication (In Progress)

🚧 Resume Score Prediction (Planned)

🚧 ATS Compatibility Check (Planned)

## Tech Stack

### Frontend

* React.js
* Tailwind CSS
* React Router
* Axios
* React Hot Toast

### Backend

* Spring Boot
* REST APIs
* Java

### AI

* Gemini API / Groq API

### Database

* MySQL
